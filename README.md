# ✈️ The Virtual Tower: DUS Live Tracker

An AI-powered Computer Vision system in development to monitor the live airfield of **Düsseldorf Airport (DUS)**. The objective is to detect aircraft, track their movement, and fuse visual data with real-time flight telemetry.

---

## 🏗 Planned Architecture

*   **Ingestion:** Capture and process live DUS video streams.
*   **Inference:** Deploy YOLO models to identify aircraft on the ground.
*   **Tracking:** Implement object tracking to maintain persistent identities.
*   **Enrichment:** Integrate ADS-B data (via OpenSky Network API) to correlate telemetry with visual detections.
*   **Visualization:** Create a dashboard for monitoring operations.
